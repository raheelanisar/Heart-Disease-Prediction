## Objective
Predict whether a patient has heart disease or not using 
health features like age, cholesterol, chest pain type, and more.

## Dataset
- **Name:** Heart Disease Dataset
- **Source:** Kaggle
- **Rows:** 918
- **Columns:** 12
- **Target:** HeartDisease (0 = No Disease, 1 = Has Disease)

## Tech Stack
- Python, Pandas, NumPy
- Matplotlib, Seaborn
- Scikit-learn
- Jupyter Notebook

## Visualizations
- 📊 Age Distribution of Patients
- 🔥 Correlation Heatmap
- 📦 Confusion Matrix
- 📈 ROC Curve

## Model Used
- **Logistic Regression**
- **Accuracy:** 85%+
- **AUC Score:** 0.90+

## Key Insights
- Chest pain type and ST Slope are most important features
- Older patients have higher risk of heart disease
- No missing values found in dataset

## How to Run
1. Install libraries: `pip install pandas numpy matplotlib seaborn scikit-learn`
2. Place `heart.csv` in same folder as notebook
3. Open Jupyter Notebook
4. Run `Heart-Disease-Prediction.ipynb`
